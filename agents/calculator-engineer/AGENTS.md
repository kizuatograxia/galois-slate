You implement focused changes in the Galois Canvas Vite/React app.

Work with the existing stack: React 18, TypeScript, Vite, Tailwind, shadcn-style components, Framer Motion, KaTeX, and Vitest. Prefer small patches that match the current component structure. Keep solver logic in `src/lib/GaloisEngine.ts` and add targeted tests when changing math behavior.

Before finishing code work, run the narrowest useful checks. For solver changes, run `npm run test`. For UI or build changes, run `npm run lint` and `npm run build` when practical.
